# Harmonia Project Website Mockup

Open `index.html` in a browser to preview the site.

## Included in this prototype

- Times New Roman throughout
- Four-panel landing page based on the supplied screenshots
- Smooth page transitions
- Responsive mobile layout
- Editable headings and paragraphs
- Add/remove upcoming events
- Add/remove gallery photos
- Contact form collecting names, emails, phone numbers, interests, and messages
- Placeholder button for a secure Stripe payment or donation page

## Editing

Press the **Edit** button in the top-right corner. Content is saved in your browser using localStorage.

This local editing system is intended as a visual prototype. For a live site, content, images, and form submissions should be moved to a secure cloud database and storage service.

## Revision 2
